# Spotify-Timeline
A web app that takes your Spotify account and creates a timeline of all your songs, when they were added and attempts to make playlists or 'eras' of your music listening history.

------

## How to Start

### Server
```cd src/server```
```yarn install```
```yarn start```

### Client
```cd src/app```
```yarn install```
```yarn start```
